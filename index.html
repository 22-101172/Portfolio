About Us

We are Seif Elislam and Omar Elhossainy — two AI students who believe that data is more than numbers, and that intelligence, when built thoughtfully, can change the way the world works.

Our journey started with curiosity — asking why machines learn, how computers see, and what language really means to an algorithm. That curiosity turned into projects. Those projects turned into a portfolio. And that portfolio is what you're looking at right now.

We work across the full spectrum of modern AI — training deep learning models that recognize human emotion from a single image, building NLP systems that breathe life into centuries-old Arabic texts, and developing recommendation engines that learn from human behavior. We don't just implement algorithms; we think about why they work, where they fail, and how to make them better.

Beyond the technical, we bring something that textbooks don't teach — genuine passion for the problems we choose to solve. This portfolio is not just a collection of work. It's a record of how we think, how we build, and where we're headed.

Featured Projects

01

Emotion Recognition using Images

An end-to-end emotion detection system that analyzes facial images and classifies emotions such as happiness, sadness, anger, and surprise with a confidence score. Built on a fine-tuned ResNet50 model with SCRFD face detection, deployed as an interactive web app using Streamlit.

Python TensorFlow/Keras ResNet50 InsightFace OpenCV Streamlit
02

Arabic NLP Chatbot: Tazkirat Dawood

A multi-stage AI pipeline built around "Tazkirat Dawood," a classical Arabic herbal medicine text spanning over 800 pages. Using Gemini's vision capabilities, the system extracts structured herb, disease, and cure data in Arabic and English, then cross-references each herb-disease pair against modern scientific literature using an AI pharmacologist agent.

Python Gemini Vision NLP Arabic Text Processing Pandas
03

PathFinder: AI-Powered Academic & Career Advisor

A full-stack AI advising system designed for university students, combining institutional data with intelligent reasoning to guide academic and career decisions. Integrates the student handbook, course catalogue, and prerequisite relationships to evaluate graduation progress, recommend course plans, and map academic paths to real-world career roles.

Python NLP RAG Database Systems REST API
